# SpeederAddon

**Firefox Extension** — A specialized tool for precise video playback rate control.

![addon_photo](./photos/image.png)

### Description

A minimalist extension designed to modify the playback rate of video elements on any webpage. The application automatically identifies the `<video>` element within the active tab and provides an intuitive interface for adjusting playback parameters.

**Key Features:**

- Precise speed adjustment with 0.1x increments (ranging from 0.1x to 5.0x).
- Instant presets for rapid switching: 2x, 2.5x, 3x, and 3.5x.
- Automated interaction with the active tab's DOM via the Scripting API.

---

### Installation Guide

To run the extension locally for development purposes, follow these steps:

1. **Clone the Repository**

   ```bash
   git clone [https://github.com/Cheengizs/SpeederAddon](https://github.com/Cheengizs/SpeederAddon)
   ```

2. **Browser Preparation**

- Open Firefox and navigate to the debugging section: `about:debugging`.
- From the left-hand menu, select **This Firefox**.

3. **Load the Extension**

- Click the **Load Temporary Add-on...** button.
- In the file dialog, select the `manifest.json` file located in the project's root folder.
